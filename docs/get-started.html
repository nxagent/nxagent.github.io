Start building with NxAgent.

This page is a minimal setup guide. Replace the placeholder package details as the framework API and distribution become final.

Install

Install the package into a Python environment.

python -m venv .venv
source .venv/bin/activate
pip install nx-agent

Project shape

A small NxAgent project can start with one app file and grow into agents, tools, and workflow modules as needed.

my-agent/
  app.py
  tools.py
  agents.py
  .env

Core ideas

  • Agent: a role-driven worker with a goal.
  • Tool: a Python function the agent can call.
  • Crew: a group of agents coordinated into a workflow.
  • Trace: structured output for observing what happened.
