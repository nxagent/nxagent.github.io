// quickstart

Your first NxAgent workflow.

Create a tiny research workflow with one tool, one agent, and one run call. This is draft API copy and can evolve with the package.

1. Create a tool

Tools are normal Python functions exposed to your agent.

from nx_agent import Tool

@Tool
def web_search(query: str) -> str:
    """Search for current information."""
    return "Search results for: " + query

2. Define an agent

Give the agent a role, goal, and the tools it can use.

from nx_agent import Agent

researcher = Agent(
    role="Research Analyst",
    goal="Find and synthesize accurate information",
    tools=[web_search],
)

3. Run the workflow

Start small, inspect the result, then add more agents or tools when needed.

result = researcher.run(
    "Summarize the latest advances in agentic AI"
)

print(result)
