// workflow

From idea to running agent.

NxAgent gives you a simple path from defining agent behavior to running and observing complete multi-agent workflows.

1. Define your agents

Give each agent a role, goal, and backstory. NxAgent uses these to guide LLM behavior and keep agents on track across long-running tasks.

2. Attach tools

Decorate Python functions with @Tool and make them available to your agents. NxAgent handles schema generation, calling, and error recovery.

3. Orchestrate with a Crew

Group agents into a Crew and define how they collaborate: sequentially, in parallel, or with dynamic task routing based on expertise.

4. Run and observe

Execute your crew and inspect every step, including agent reasoning, tool calls, memory reads, and final outputs.
